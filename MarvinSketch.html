JSMarvinSketch v 1.0 beta

Use JS MarvinSketch to draw or edit a molecule. The resulting SMILES or InChI string may be used to search for matching molecules in the PDB Chemical Component Dictionary.

Note that matches will include any chemical component in the dictionary, including polymeric ones like Alanine or Adenosine.
